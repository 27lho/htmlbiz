Mystic Connections — Static Starter

Simple responsive static site scaffold for the Mystic Connections crystal shop.

Preview locally:

```bash
# from the project root
python3 -m http.server 8000
# then open http://localhost:8000
```

What's included:
- `index.html` — Home
- `gallery.html` — Gallery
- `pricing.html` — Pricing
- `contact.html` — Contact
- `css/styles.css` — Styling
- `js/main.js` — Small interactions (lightbox)
- `assets/` — logo + SVG placeholders

Customize copy, replace SVG placeholders with real photos, and add server/backend if required.
